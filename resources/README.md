# Native assets

Place two source files here before running `npx @capacitor/assets generate`:

- `logo.png` — 1024×1024 app icon (opaque or with transparency)
- `splash.png` — 2732×2732 splash image, subject centered

The current app logo lives on the CDN — download it and save as `logo.png`:

```
curl -o resources/logo.png https://<your-lovable-domain>/__l5e/assets-v1/0d7f4e2c-a1f8-4385-b015-e30b1919824e/uuc-logo.png
```
